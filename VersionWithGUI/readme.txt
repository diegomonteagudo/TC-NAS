

NAS Project : generate GNS3 router config files from a json intent file
Uses IPv4, OSPF, BGP and BGP/MPLS VPN


How to use :

(1. Create a GNS3 project, add the routers, name them and link them together)
(2. Make a .json intent file (an example is provided))

3. Launch main.py
4. Provide the path for the intent file and the .gns3 project file
5. Produce the config files with "Produce"
6. Move them to the GNS3 projectt with "Move"
